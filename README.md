<h1 align="center">Python Template</h1>

**SQL Python 정규과제 템플릿**입니다.  
이 템플릿은 다트비 7기에서 매주 주어지는 SQL Python 정규 과제를 정리하고 제출하기 위한 기본 템플릿을 제공합니다.  

---

## 학습 목표
- **파이썬 기초 문법과 NumPy·Pandas의 핵심 구조를 이해**하고 데이터 분석의 프로그래밍 기반을 확립합니다.
- **다양한 데이터 소스의 수집·저장 및 전처리 과정을 학습**하여 분석 가능한 데이터 환경을 구축합니다.
- **조인, 병합, 그룹 연산 등 데이터 재구성 및 집계 기법을 활용**해 구조화된 분석 데이터를 생성합니다.
- **시각화 및 시계열 처리 기법을 적용**하여 데이터 해석과 인사이트 도출 역량을 강화합니다.

---

## 📂 제출 방법 (리마인드)
1. 이 템플릿 레포지토리의 `assignment` 폴더 내의 마크다운 파일을 개인 계정 레포지토리에 복사합니다.  
2. 매주차 과제를 `weekX/README.md`에 정리합니다. (형식은 자유)  
3. 과제가 완료되면 **개인 레포 링크**를 과제 제출 시트에 업로드합니다.  
> (이미지 폴더는 개인 레포에 따로 만들어서 저장을 해야 업로드가 됩니다.)
